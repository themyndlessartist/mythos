# ADR-013 — Shared World Rules

ADR number: ADR-013

Title: Shared World Rules

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

Mythos aims for players and NPCs to follow the same world rules whenever practical.

## Decision

The framework will prefer shared systems for player and NPC capabilities when approved design permits.

## Consequences

System specifications must identify where shared rules apply and where player-specific or NPC-specific exceptions are necessary.

## Related systems

Character Framework, NPC Framework, Economy Framework, Organization Framework
