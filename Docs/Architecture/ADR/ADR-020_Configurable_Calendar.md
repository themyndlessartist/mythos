# ADR-020 — Configurable Calendar

ADR number: ADR-020

Title: Configurable Calendar

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

Different Mythos titles may require different calendars, day lengths, seasons, and time terminology.

## Decision

The framework will support configurable calendars rather than assuming one universal setting-specific calendar.

## Consequences

Time specifications must define calendar configuration, persistence, display, and simulation integration.

## Related systems

Time Framework, Content Packages, Region Framework, Simulation Systems
