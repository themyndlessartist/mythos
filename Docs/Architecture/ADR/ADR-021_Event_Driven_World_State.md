# ADR-021 — Event-Driven World State

ADR number: ADR-021

Title: Event-Driven World State

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

World changes should be traceable and capable of driving simulation, history, and consequences.

## Decision

The framework will favor event-driven world-state changes for significant actions and simulation outcomes.

## Consequences

Event specifications must define event identity, ordering, routing, persistence expectations, and integration with affected systems.

## Related systems

Event Framework, World History, Save Systems, Simulation Systems
