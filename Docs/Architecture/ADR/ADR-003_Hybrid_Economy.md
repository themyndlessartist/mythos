# ADR-003 — Hybrid Economy

ADR number: ADR-003

Title: Hybrid Economy

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

The Mythos economy should feel continuous and believable without requiring every routine transaction to be simulated in full detail.

## Decision

The framework will use a hybrid economy model, allowing important economic activity to be simulated while routine production, inventory, and exchange may be abstracted.

## Consequences

Economy specifications must define which activities require detailed simulation and which may use abstraction while preserving believable outcomes.
