# ADR-008 — Hybrid World Simulation

ADR number: ADR-008

Title: Hybrid World Simulation

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

The world must feel persistent while remaining technically scalable.

## Decision

The framework will use hybrid world simulation, with detailed local simulation and abstract distant simulation.

## Consequences

Simulation specifications must define fidelity tiers, transition rules, and consistency expectations between local and distant world state.

## Related systems

Region Framework, Time Framework, NPC Framework, Event Framework
