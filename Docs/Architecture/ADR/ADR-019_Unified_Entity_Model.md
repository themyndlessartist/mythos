# ADR-019 — Unified Entity Model

ADR number: ADR-019

Title: Unified Entity Model

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

Mythos requires reusable representation of people, places, organizations, assets, and other persistent world objects.

## Decision

The framework will use a unified entity model conceptually, while the detailed implementation model remains undecided.

## Consequences

Entity specifications must define identity, ownership boundaries, relationships, and extension points without selecting ECS, OOP, or another final implementation architecture yet.

## Related systems

Entity Framework, Character Framework, Region Framework, Organization Framework
