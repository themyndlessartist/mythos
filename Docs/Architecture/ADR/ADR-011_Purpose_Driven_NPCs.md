# ADR-011 — Purpose-Driven NPCs

ADR number: ADR-011

Title: Purpose-Driven NPCs

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

Named NPCs should be citizens with identity and purpose rather than decorative entities.

## Decision

The framework will support named NPCs with persistent identity, purpose, occupation, schedule, relationships, traits, goals, and ambitions at an appropriate simulation depth.

## Consequences

NPC specifications must define simulation tiers and data requirements without requiring every NPC to be simulated at maximum detail.

## Related systems

NPC Framework, Character Framework, Relationship Framework, Time Framework
