# ADR-015 — Time Pause Rules

ADR number: ADR-015

Title: Time Pause Rules

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

Interface screens may pause time, while normal world interactions should generally allow time to continue.

## Decision

The framework will distinguish between time-pausing interfaces and time-progressing world interactions.

## Consequences

Future specifications must define pause categories, player-facing rules, and effects on simulation layers.

## Related systems

Time Framework, UI Systems, Event Framework, Simulation Systems
