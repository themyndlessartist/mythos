# ADR-017 — Hierarchical Regions

ADR number: ADR-017

Title: Hierarchical Regions

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

World simulation requires spatial organization that can scale from local areas to larger political or geographic structures.

## Decision

The framework will support hierarchical regions as a foundational world-organization concept.

## Consequences

Region specifications must define parent-child relationships, simulation scope, event routing, and persistence boundaries.

## Related systems

Region Framework, Event Framework, Time Framework, Simulation Systems
