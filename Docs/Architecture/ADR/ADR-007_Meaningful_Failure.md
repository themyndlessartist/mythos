# ADR-007 — Meaningful Failure

ADR number: ADR-007

Title: Meaningful Failure

Version: 0.1

Status: Approved

Owner: Mythos Executive Development

Date: July 2026

## Context

Failure should be possible and should create new gameplay situations rather than automatically ending the experience.

## Decision

The framework will support persistent consequences for failure where approved systems define them.

## Consequences

Specifications must account for recoverable setbacks, changed world state, and consequences that can become future gameplay inputs.

## Related systems

Event Framework, Character Framework, Economy Framework, Relationship Framework
